# 农业农村部农业信息化标准化技术委员会2025年度年会在北京召开

> 来源: [农业农村部智慧农业-标准规范](https://www.moa.gov.cn/ztzl/zhnyzxd/bzgf/bzzj/202601/t20260128_6481070.htm)  
> 日期: 2026-01-28
> 摘要: 2026年1月27日,农业农村部农业信息化标准化技术委员会(以下简称“农信标委”)2025年度年会在京召开,市场与信息化司有关负责同志和农信标委全体委员参加会议。
会议强调,要深刻认识新时代农业信息化标准化工作的重要意义,认真贯彻落实韩俊部长在农业农村标准化工作推进会上的部署要求,准确把握农业信息化标准化工作面临的新形势与新任务。
...

会议强调,要深刻认识新时代农业信息化标准化工作的重要意义,认真贯彻落实韩俊部长在农业农村标准化工作推进会上的部署要求,准确把握农业信息化标准化工作面临的新形势与新任务。

会议要求,要以务实举措推进农业信息化标准化工作,一是加快制定产业急需标准,系统谋划“十五五”期间智慧农业标准制定计划;二是健全标准全过程质量管理机制,强化从立项到实施的全流程管控;三是加强标委会自身建设,优化工作机制,压实委员责任,建设高水平标准化专业队伍。

会议审议并通过了《农信标委2025年度工作报告》《委员增补建议》和《设立研究组建议》,与会委员围绕智慧农业标准化深入研讨、建言献策,中国标准化研究院、江苏大学专家分别围绕“农业国家和行业标准申报审核要点”“人工智能赋能农业的实践与标准化挑战”作专题辅导。
